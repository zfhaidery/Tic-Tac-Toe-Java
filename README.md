# Tic Tac Toe Game in Java

A simple, fully functional **console-based Tic Tac Toe** game built in Java for two players (Player X and Player O).  
The game includes proper win detection, draw handling, input validation, and clean board display.

## Features
- 3Ã—3 classic Tic Tac Toe board
- Two-player mode (X starts first)
- Input validation (only 1-3 allowed, no overwriting filled cells)
- Clear win/draw messages
- Beautiful formatted board in console with grid lines
- Error handling for invalid inputs

## How to Run
1. Make sure you have **JDK 8 or higher** installed
2. Clone or download the repository
3. Open terminal / command prompt in the project folder
4. Compile the code: javac TicTacToe.java
5. again write:java TicTacToe
   your code will run and can enjoy the game.
